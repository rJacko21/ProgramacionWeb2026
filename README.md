# Portfolio Web - Sagas Gaming

Un portfolio web responsivo y moderno que presenta las sagas de **Resident Evil** y **The Legend of Zelda**, con soporte para tema oscuro/claro y navegación intuitiva.

## 📋 Características

✅ **Diseño Responsivo** - Se adapta perfectamente a dispositivos de cualquier tamaño (desktop, tablet, móvil)

✅ **Tema Oscuro/Claro** - Botón para cambiar entre temas con persistencia en localStorage

✅ **Menú Desplegable** - Navegación con dropdown menus intuitivos

✅ **Menú Hamburguesa** - Menú responsivo para dispositivos móviles

✅ **Scroll Suave** - Navegación fluida entre secciones

✅ **Secciones Detalladas** - Información completa sobre:
   - Resident Evil: Historia, consolas, detalles técnicos
   - The Legend of Zelda: Historia, consolas, detalles técnicos

✅ **Footer Completo** - Datos de contacto, información adicional y apartado legal

## 📁 Estructura de Archivos

```
html/
├── index.html      (Estructura HTML principal)
├── styles.css      (Estilos CSS responsivos y temas)
└── script.js       (Interactividad JavaScript)
```

## 🚀 Cómo Usar

1. **Abrir en el navegador:**
   - Haz clic derecho en `index.html`
   - Selecciona "Abrir con" → Navegador web de tu preferencia
   - O copia la ruta: `file:///c:/Users/admin/Documents/ProgWeb2026/html/index.html`

2. **Navegación:**
   - Usa el menú superior para desplazarte entre secciones
   - En móviles, el menú se convierte en un menú hamburguesa
   - Haz clic en "Sagas" para ver el menú desplegable

3. **Cambiar Tema:**
   - Haz clic en el botón 🌙 (claro) o ☀️ (oscuro) en la esquina superior derecha
   - El tema se guarda automáticamente en tu navegador

## 📱 Responsividad

- **Desktop (> 768px):** Menú horizontal completo
- **Tablet (768px):** Menú adaptado con hamburguesa
- **Móvil (< 480px):** Interfaz completamente optimizada para touch
- **Muy pequeño (< 320px):** Ajustes adicionales para pantallas mínimas

## 🔧 Funcionalidades JavaScript

- **Cambio de tema:** Guardado en localStorage
- **Menú responsivo:** Se adapta según el tamaño de la pantalla
- **Menú desplegable:** Funciona en desktop (hover) y móvil (click)
- **Animaciones:** Transiciones suaves al desplazarse
- **Sincronización de pestañas:** El tema se sincroniza entre pestañas abiertas
- **Navegación activa:** Indica la sección actual mientras se desplaza
